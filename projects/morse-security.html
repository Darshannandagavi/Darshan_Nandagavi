1. Project Summary

Morse Decrypter lets users paste or input Morse code (dots, dashes, slashes and spacing) or upload short audio snippets and receive decoded plain text. The repository contains a React client (client/) and a minimal Node/Express server (server/). The client performs most UI decoding but can call the server for audio processing or alternative decoding strategies.

  • Server fallback: POST /api/decode-text -> server returns decoded plaintext
